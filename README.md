# Prueba de SQL

## **Objetivo**
* Realizar una prueba de SQL, donde vamos a demostrar las habilidades en codigo, para hacer la creacion de una tabla y posterior a eso realizar un EDA(Exploratory Data Analyst).

## **Descripción**
* Este codigo va desde el comienzo en la creacion de una tabla, hasta tener una analisis exploratorio de los datos.

## **Qué voy a Aprender?** 
Al final de esta data seremos capaces de:
* Crear una tabla en SQLITE
* Crear analisis descriptivos, y exploratios
 
## **Servicios a Utilizar**
* [SQL LITE] https://sqliteonline.com/

A continuación voy a explicar detalladamente el código del archivo SQL
